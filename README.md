# Movie App (JavaScript ver.)

OMDb API를 활용해 VanillaJS 영화 검색 애플리케이션 예제로 아래 명시된 기술을 활용했습니다.

- OMDb API
- 구글 폰트
- Vercel Hosting / Serverless Functions / 환경변수


### 프로젝트 시작하기

```bash
$ npm i
$ npm run vercel
```
